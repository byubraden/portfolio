# Step 1: Project Setup & Configuration

## Goal
Get the project scaffolded, dependencies installed, Tailwind configured with the dark/blue theme, and the dev server running.

## Tasks

- [ ] Initialize Vite + React project (`npm create vite@latest`)
- [ ] Install dependencies:
  - tailwindcss (v4)
  - framer-motion
  - react-icons
  - react-scroll
- [ ] Configure Tailwind with custom theme colors:
  - Dark backgrounds: slate-900 / slate-950
  - Primary blue: blue-500 / blue-600
  - Accent: blue-400
  - Cards/surfaces: slate-800
- [ ] Set up folder structure:
  - `src/components/`
  - `src/data/`
  - `src/assets/`
  - `src/assets/projects/`
- [ ] Set up `index.html` with:
  - Page title
  - Meta description
  - Favicon placeholder
  - Open Graph meta tags (placeholder values)
- [ ] Verify `npm run dev` starts and shows a blank page with dark background
- [ ] Create initial `ai/changelog.md` entry

## Done When
- Dev server runs without errors
- Page loads with dark (slate-950) background
- All dependencies are installed
