# Step 4: Projects, Contact & Footer

## Goal
Build the projects showcase, contact section, and footer to complete all page sections.

## Tasks

### Projects
- [ ] Create `src/data/projects.js` with placeholder project data:
  - title, description, tech stack array, GitHub link, live link, image path
  - 3-4 placeholder entries (user will replace with real data)
- [ ] Create `Projects.jsx` component
- [ ] Section heading: "Projects"
- [ ] Responsive card grid (1 col mobile, 2 cols tablet/desktop)
- [ ] Each card includes:
  - Project image/screenshot (placeholder if not provided yet)
  - Title
  - Description (2-3 sentences)
  - Tech stack tags (small pills/badges)
  - Links to GitHub and/or live demo
- [ ] Hover effects on cards (subtle scale, shadow, or border glow)

### Contact
- [ ] Create `Contact.jsx` component
- [ ] Section heading: "Get In Touch"
- [ ] Brief encouraging message (1-2 sentences)
- [ ] Prominent LinkedIn button/link (opens in new tab)
- [ ] Styled consistently with the rest of the site

### Footer
- [ ] Create `Footer.jsx` component
- [ ] Copyright notice with current year
- [ ] Social icon links: GitHub, LinkedIn
- [ ] Clean, minimal design

## Done When
- Projects section shows 3-4 cards with hover effects
- Contact section has a working LinkedIn link
- Footer displays at the bottom with social links
- All sections responsive on mobile and desktop
