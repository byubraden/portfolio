# Step 5: Animations, Responsiveness & Final Polish

## Goal
Add entrance animations, do a full responsive pass, finalize SEO tags, and verify the production build.

## Tasks

### Animations (Framer Motion)
- [ ] Add fade-in / slide-up entrance animations to each section
- [ ] Trigger animations when sections scroll into view
- [ ] Keep animations subtle and fast (no slow/flashy effects)
- [ ] Ensure animations don't cause layout shift

### Responsive Design Pass
- [ ] Test every section at mobile width (~375px)
- [ ] Test every section at tablet width (~768px)
- [ ] Test every section at desktop width (~1280px)
- [ ] Fix any overflow, text wrapping, or spacing issues
- [ ] Verify hamburger menu works correctly on mobile
- [ ] Verify all tap/click targets are large enough on mobile

### SEO & Meta
- [ ] Final page title in `index.html`
- [ ] Meta description
- [ ] Open Graph tags (og:title, og:description, og:image, og:url)
- [ ] Favicon (replace placeholder with real one, or use a simple generated one)

### Final Verification
- [ ] `npm run build` succeeds with no errors
- [ ] All nav links scroll to correct sections
- [ ] Resume PDF download works
- [ ] All external links (LinkedIn, GitHub) open in new tabs
- [ ] No console errors or warnings
- [ ] Page loads quickly (no large unoptimized assets)

## Done When
- Site feels polished with smooth animations
- Looks good on all screen sizes
- Production build passes
- All links and interactions work correctly
- Ready for deployment
